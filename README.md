## 📋 Sobre o Projeto  

Uma calculadora simples desenvolvida em **C#** utilizando **Windows Forms**.  

O projeto tem como objetivo praticar conceitos de **lógica de programação** e desenvolvimento de aplicações desktop, oferecendo uma interface gráfica intuitiva para realizar operações matemáticas básicas.

---

## ✨ Funcionalidades  

- ➕ Adição  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🖥️ Interface gráfica simples e fácil de usar  

---

## ▶️ Como Executar  

1. Abra o projeto no **Visual Studio**  
2. Restaure as dependências (se necessário)  
3. Execute o projeto pressionando `F5` ou clicando em **Iniciar**

---

## 🛠️ Tecnologias Utilizadas  

- C#  
- .NET Framework / .NET  
- Windows Forms  

---

## 📌 Objetivo  

Este projeto foi desenvolvido com fins educacionais, com foco em:  

- Prática de lógica de programação  
- Manipulação de eventos em interfaces gráficas  
- Desenvolvimento de aplicações desktop
